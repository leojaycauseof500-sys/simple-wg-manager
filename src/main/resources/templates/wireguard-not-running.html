操作

refresh 重新检查状态 login 前往登录 terminal 显示启动命令
code 启动 WireGuard 服务 
sudo wg-quick up wg0
将 "wg0" 替换为您的接口名称（如果不同）。 此命令需要管理员权限。
code 检查服务状态 
sudo wg show
显示当前 WireGuard 接口和客户端状态。
code 停止服务 
sudo wg-quick down wg0
停止 WireGuard 服务。
warning 故障排除
  • check_circle 确保 WireGuard 内核模块已加载：lsmod | grep wireguard
  • check_circle 检查配置文件是否存在：ls /etc/wireguard/
  • check_circle 验证配置文件语法：sudo wg-quick strip wg0
  • check_circle 查看系统日志：sudo journalctl -u wg-quick@wg0
  • check_circle 检查防火墙设置，确保 UDP 端口 51820 已开放
